Aqui, serão expostas as minhas notas trimestrais, para que eu possa treinar a mecânica das tabelas em HTML, e, de quebra, me organizar melhor com minhas notas escolares. :)

Lembrando que todos os elementos deste website são apenas para estudos e teste.Portanto, não considere e nem leve nada a sério do que existir nesse lugar.

Aproveitando para lembrar também para lembrar que, no código, a organização de células está na sintaxe x.y, primeiro horizontal e depois vertcal.

Obrigado pela atenção!

Para criar uma tabela utilizamos a tag table para criar a estrutura, a tag tr para criar cada linha e td para criar cada célula de conteúdo.
